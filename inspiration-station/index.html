📊市面竞品现状
🔥
抖音创作者中心「热点宝」
平台官方
  • 基于热点榜单推荐话题，强在平台官方数据
  • 更偏数据看板，不直接把"不知道拍什么"转成可执行选题
  • 缺少选题资产管理，看完数据仍需人工整理
🤖
Coze / GPTs 类选题 Bot
AI 平台
  • 通用 AI 生成选题，可自定义 Prompt 和工作流
  • 无垂类/阶段区分，千人一面
  • 需要自己搭建，普通创作者无法开箱即用
🎬
剪映「图文/脚本生成」
制作工具
  • 强在制作执行（AI 文案、自动剪辑），移动端体验好
  • 不解决"今天该拍什么"的选题决策问题
  • 选题推荐偏单次生成，无选题库沉淀
📈
飞瓜 / 蝉妈妈
数据平台
  • 给大盘数据（达人榜、传播指数、竞品涨粉分析）
  • 用户要自己从数据推导"拍什么"，不直接生成选题
  • 价格和学习门槛偏高，个人创作者使用压力大
💡我们的创新点
🎯
四维个性化输入
垂类 × 账号阶段 × 受众画像 × 卖点，生成的选题是「只属于你」的，而非千人一面
📊
输出分层策略
高潜力 / 差异化 / 长尾三类选题，冷启期和成长期的推荐完全不同
🔍
爆款解构器
拆解共鸣点 + 可迁移框架 + 同垂类变体，让创作者知道为什么火、怎么学
📚
选题库管理
标记已拍/待用/不适合，从「一次性推荐」变成「可复用选题资产」
⚔️
内置竞品对比视角
展示对抖音创作者中心、Coze、剪映等的调研能力与差异化定位
核心判断
个人创作者缺的不是「更多选题列表」，而是「基于我这个账号现在的阶段，今天该拍什么」。所以我们不拼数据规模，而是把账号阶段感知 + 选题决策产品化。
🚀后续优化方向
接入实时热点 API

同垂类爆款结构参考
加入近 7 天同垂类爆款的内容结构拆解，让选题更可执行

链路数据串联
